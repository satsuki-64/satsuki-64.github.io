12.1 法律中的事件变量

死刑是否重判？

著作权保护

  • 中国五十年保护
  • 美国七十年保护

时间并不是唯一的因素，时间提供了一个预期；

  • 为什么有些法律不追诉？
    • 有些问题的起诉时间限制，可以让人们对于未来的安排更有预期；
    • 并且很多证据会逐渐消失，取证困难；
      • 集中成本，成本降低；

时间是用来确定权利义务的标准；

12.2 法律中的时间维度

基本原则：不追诉立法之前的案件

  • 法律不溯及既往
    • 但是其实是不可能的
    • 立法的主要目的：引导人们的行为，因为知道法律的规定，因此可以更好的保护人们的预期；
  • 此法律的所需前提
    • 需要人们在这之前知道这个法律；
      • 许多法律都会有知情的问题：犯法者是否在发生之前，知道这条法律；
    • 因此法律在裁决的时候还是要溯及既往；
  • 因此：只要法律涉及修改，就一定会涉及既往；
    • 因此：法律应该尽可能保持稳定，因为法律经常修改，将会影响人们的预期；
      • 不是看到有法律有坏处，就要去修改它；因为有可能修改本身带来的坏处，甚至都会大于其带来的好处

法律与稳定

  • 尽量少修改法律 -> 试图使得法律在长时间内保持稳定；
  • 法律的目的 -> 使现状神圣化；
    • 只要有现状基础的规则能够随着时间推移，就能形成具体的规则的形式，进而使得现状神圣化 -> 法律、规则;
  • 法律的时间性：法律和时间、空间总是无法分割的；
    • 抽调了时间的变量，法律就没有了；
    • 法律的形成需要时间的维度，并依靠时间逐渐得到神圣化；

为什么中国思考法律的时候，为什么经常往来时间的问题？因为中国的学者经常会先入理论的维度，去除掉时间、实践的部分

  • 理论化的思考问题和具体的思考问题
    • 理论化的思考问题：把情况从上下文中抽离出来，将符号之间进行组织、推理，最后可能得出不符合时间上下文的结论；
  • 具体的、时间化的思考
    • 任何的问题需要补充足够的上下文，尤其思考线性的时间本身所带来的变化；

法律与时间的关系

  • 革命、改革，经常会和法律产生冲突；
    • 法律与政治的关系 -> 法律经常会和社会的变革产生冲突；
    • 打破这些常规的制度 -> 原本预期在的权利，现在需要被限制；
      • 比如奥运会期间限行动；
      • 赈灾时期的临时法律制度；
  • 现状 -> 现代中国需要变革，因此从现状来讲，需要破坏法律；
    • 从法制的角度 -> 法律的时间维度与政治改革的冲突，政治家需要维持这两者的平衡；
    • 比如如果没有三十年的无法律、但是改革的时期，现在很多法律根本就不存在存在的前提；
  • 因此，为什么法制很难
    • 因为在这之前，改革、发展是第一位的，而这些东西本身会使得法律无法稳定下来；
    • 因此中国的改革，总是要一个地方先进行实验，然后再推广；
    • 只有在长时间稳定的政权下，才有能力证明一个制度的好坏；
    • 制度是需要时间的；
  • 人们对于幸福的享受，很多时候还是来自于过去的稳定以及经验，以及预期
    • 人不是说都是喜欢创造性的、新奇的生活，而是先有稳定的、保守的生活，然后再有的在这之上的细微的创新、新奇；

改革和法制的关系

  • 任何形式的变革，都会导致一小部分的人的权力、利益受到一定的侵害、减少；
  • 所以现在的改革是否进行，实质上是平衡现有的既得利益，和改革带来的发展之前冲突、矛盾的问题；
    • 曾经需要改革，改革完之后部分人获得了工作，成为了既得利益者，然后是否要再对他们进行改革？还是保持当前的稳定？

12.3 中国法制变革的时间因素

时代的因素

  • 中国的改革是被全球化、经济全球化被迫推动的改革；
  • 而且这种改革要快速达成：一万年太久，只争朝夕；

法制的理论逻辑和法制的实践逻辑完全分离

  • 实践都是需要时间的、实践都是需要看到后果的、证明的；
  • 但思想是只需要逻辑推理就行的，只需要小部分知识分子快速就可以构建出来；
    • 理论家的思考是不需要时间的；
    • 但实践是需要大量人的血泪出来的；
  • 法制是一整个民族的实践的事业；

中国社会甚至不给那么多时间来对法制进行验证

心态问题：人类心态的保守倾向

  • 整体保守的，人对于变化的承受力是有限的；
  • 预期问题：变化太快时，会打破预期，人会产生不适应 -> 会对意义产生消失；
    • 改革时期人的心态变化太大；
    • 就算人的提升很大也一样；
