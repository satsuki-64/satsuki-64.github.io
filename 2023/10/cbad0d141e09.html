查找
查找的两种实现：

  • 对每一个 Value 进行递归
  • Stored items sorted increasing by key allows:
    • faster find min/max (at first and last index of array)
    • faster finds via binary search: O (log n)；

对于两种实现的 Set 的结论的效率对比

  • Sorted 是已经经过了排序的数组；
    • build 的时候会花费更多的时间；
    • 但是查找元素会更快；
  • Sorted 可以提供查找的效率

3.2 对于查找的实现：Permutation Sort

对于输入以及输出的描述

  • Input:
    • (static) array A of n numbers
  • Output:
    • (static) array B which is a sorted permutation of A
    • Permutation: array with same elements in a different order
    • Sorted: B[i − 1] ≤ B[i] for all i ∈ {1, . . . , n}
  • 两个需要知道的概念
    • Destructive：Overwrites the input array；
    • In Place：Use O (1) extra space；

什么是 Permutation Sort（置换排序）
具体步骤：

  • 第一步：枚举当前元素
