2.2 基于数组的数据结构

2.2.1 Static Array

Key：Word RAM Model Of Computation；
可以随机访问磁盘中的数据，并且速度十分快；

  • Array ：连续的在内存当中的排列；
  • 访问 Array：和访问 Memory 的数组基地址 + i 偏移地址；
  • 时间复杂度
    • 访问：O (1)
    • 遍历：O (n)
  • 空间复杂度（假设）：内存分配时间
    • 分配 n 大小的数组，花费 O (n)的时间；
    • Space = O (TIme)；
