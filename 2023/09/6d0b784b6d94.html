3.3.1 抽象 Window 类实现

Windows 类解析
此类的作用是用于被其他具体的平台实现，因此都是抽象函数或者函数指针，并不实际拥有数据；

  • 回调函数
    • 在 Windows 类当中，有一个 EventCallbackFn 函数指针；
  • Create 函数
    • 等待被实现，用于在不同的平台创建 Window ；
  • 其余都是抽象接口，等待被实现

WindowProps 类
用于存储和 Window 相关的属性数据；

  • 标题
  • 高度、宽度
